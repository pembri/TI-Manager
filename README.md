# TI-Manager: Theme and Icon Installer for Linux

**TI-Manager** is a tool designed to simplify the installation of themes and icons packaged in archive formats such as `.zip`, `.tar.gz`, and `.tar.xz` on Linux. The tool automatically extracts and configures the theme or icon files and places them in the appropriate system directories, supporting popular Linux desktop environments such as GNOME, KDE Plasma, XFCE, and more.

## Key Features

- **Support for Various Archive Formats**: Supports popular archive formats like `.zip`, `.tar.gz`, `.tar.xz`, and others.
- **Automatic Installation**: Automates the process of extracting and placing theme or icon files in the correct system directories.
- **Desktop Environment Compatibility**: Supports installation on various Linux desktop environments, including GNOME, KDE Plasma, XFCE, and others.
- **User-Friendly Interface**: Designed to be easy to use, making it suitable for both beginner and advanced users.

## How to Use

1. **Download the Latest Release**
   Visit the [Releases Page](https://github.com/pembri/TI-Manager/releases) to download the latest release of TI-Manager.

2. **Install the Application**
   Once you have downloaded the `.deb` package, you can install it using either `gdebi` or `dpkg`:

   - **Using gdebi (Recommended)**:
     ```bash
     sudo gdebi TI-Manager-version.deb
     ```
   
   - **Using dpkg**:
     ```bash
     sudo dpkg -i TI-Manager-version.deb
     ```
   
3. **Run the Application**
   Once the installation is complete, you can run TI-Manager from your application menu or by typing the following command in the terminal:
   ```bash
   ti-manager
